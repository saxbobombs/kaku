# Aufgabenergebnis

## Eingesetzte Technologien / Frameworks

Folgende Technolgien / Frameworks setze ich in meinem Projekt ein:

- Vue.js 2

Vue.js ist eins der verbreitesten Frontend-Frameworks weltweit. Gerade in Bezug auf Erweiterung, Kompatibilität mit modernen Browsern und bereits bestehenden Features bietet das Framework eine gute Basis für Single Page Anwendungen. PixelArt wurde so erstellt, dass sie sich auch in andere Vue.js-Projekte integrieren lässt.

## Installation / Ausführen des Projektes

Folgende Komponenten müssen lokal installiert sein:

- [nodejs](https://nodejs.org/en/) v14.13.1

Um das Projekt lokal auszuführen, folgendes in der Commandline / Bash eingeben:

```console
$ git clone https://github.com/saxoneee/pixelart.git pixelart
$ cd pixelart
$ npm install
$ npm run serve
```

Danach Browser öffnen und zur Url http://localhost:8080/ navigieren.

Auf github-pages steht auch ein Build zur Verfügung: https://saxoneee.github.io/pixelart/dist/
