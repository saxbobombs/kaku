# kaku

draw something

https://saxbobombs.github.io/kaku/dist/

## used technologies

* Vue.js 2
* @duoa/vue-color-picker
* Bootstrap
* BootstrapVue
* downloadjs
* bowser
* Font Awesome 5

## dev

* run `npm install` once
* run `npm run serve`
* open http://localhost:9001/

## build

* run `npm run build`
